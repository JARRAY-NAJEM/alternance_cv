EJRILI YAMMI

With the contribution of Pristini Knowledge Group we have created a mobile application - EJRILI YAMMI - with a chatbot for virtual assistance in emergency situations. We designed and developed a chatbot to provide accurate information and support during emergencies. The training covered mobile application development, user experience design, and the use of technologies such as Flutter, Flask, natural language processing (NLP) and machine learning.

AI-ScanCOVID

Creating a platform for a project focused on classifying COVID-19 using an image scanner.. It accurately labels scanned images as healthy, infected with COVID-19, or showing viral pneumonia. The training covered developing a web application using techniques such as Angular, Django, Flask and machine learning.

Sign Now

During my summer internship with ETC Tunisie, I had a wonderful opportunity to develop a web application focusing on electronic document signatures using a combination of Angular, Node.js, API, and MySQL. The web application I developed offers a seamless electronic signature experience, allowing users to securely sign and manage documents online. It eliminates the need for physical paperwork, saving time and resources.
